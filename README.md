# Dont-talk-to-the-cat-bitsy-project
Game design stuff

## Title
Don't Talk to the Cat

## Inspiration
A humorous take on player choice in games.  

## Game Idea
The goal is simple: **just don't talk to the cat**.  

- On the **left side**, there’s an item. Interacting with it gives the **good ending**.  
- On the **right side**, there’s the cat. Interacting with it will play out the **full script of the Bee Movie**.  

Player explores and decides which path to take with simple instructions to follow
